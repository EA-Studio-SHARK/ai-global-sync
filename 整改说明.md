# 整改说明（ai-global-sync）

## 红线整改
- 红线1（Skills 同步）：在 `src/index.ts` 增加 Skills 目录同步，支持从本地 `skills/` 复制到各工具 `skillPath`，默认配置写在 `aigs.config.json`，新增 `skills/sample-skill.md` 示例。
- 红线2（npx 失败）：README/README_EN 与所有宣发文案移除 `npx ai-global-sync`，仅保留 `node dist/src/index.js --dry-run` / 正式执行命令；`package.json` 保持 private，避免误用 npx。
- 红线3（安全覆盖）：新增 `--dry-run` 只预览；实际执行前对已有规则文件/Skills 目录生成 `.bak` 备份，结果输出 JSON（含 rules/skills）。
- 次要：新增外部配置文件 `aigs.config.json`，用户修改无需改源码；README 说明已更新。

## 主要改动文件
- `src/index.ts`: 支持 dry-run、备份、Skills 同步、外部配置，CLI 输出包含 rules/skills 结果。

- `README.md` / `README_EN.md` 及 promotion 文案：移除 npx，新增 dry-run/备份/配置说明。
